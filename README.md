# ECE 227 Final Project

## Installation

Install [pixi](https://pixi.sh/latest/#installation)

You can then do `pixi install` to install all the dependencies required for this project.
To run, please do `pixi run data` to generate all relevant data files, then run `pixi run main` to run the program.

## Generate Data

To grab the data:
```bash
pixi run data
```

This will generate a `social.txt`, `comm.txt`, and `collab.txt` in the `data/` directory.

## Run the Project
```bash
pixi run main
```
